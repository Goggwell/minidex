Documentation

To use Minidex locally, you can clone the repository:

→ mkdir minidex
→ cd minidex
→ git clone https://github.com/Goggwell/minidex.git

If you are using VSCode, you can start the project with Live Server.
If not, you can simply do npm run watch or yarn watch:

→ # starts server on http://localhost:3000
→ yarn watch
